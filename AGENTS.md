# Repository Instructions for Codex

- Keep this project as a browser-playable HTML5 canvas game.
- Use only original, code-generated geometric visuals.
- Do not add copyrighted assets, sprites, external images, music, or libraries.
- Make small, focused changes that are easy to review.
- Keep code readable and commented where comments clarify behavior or intent.
- Do not rewrite the entire project unless explicitly asked.
- After each task, explain what changed and how to test it.
- Use Codex's built-in pull request tool, not terminal `git push` or `gh`.
